Lista ordenada

uma lista ordenada serve para listar itens ou algma coisa de forma ordenada, crescente ou decrescente, colocando número de ordem. Segue abaixo um exemplo:

Lista ordenada

  1. peixe frito
    • pacu
    • tambaqui
    • pirarucu
  2. feijão
    • preto
    • badu
    • rajado
  3. frango frito
  4. carne assada
    • picanha
    • alcatra
    • bisteca
    • costela

Lista desordenada

Uma lista desordenada não tem ordem de privacidade
